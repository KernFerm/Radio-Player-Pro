Radio Player Pro is a Windows desktop app for browsing, searching, and streaming internet radio stations with favorites, local settings, and a desktop-friendly player experience.

## What It Does

- 🔎 Search stations by name, tag, country, and language
- 🌍 Browse a large live station catalog
- ❤️ Save favorite stations for quick access
- ▶️ Restore your last played station and playback settings
- 🎵 Use the mini player while listening
- 💬 Open Help and support links directly from inside the app

## Features

- 🖥️ Fast desktop layout with a resizable sidebar
- ⚡ Smooth scrolling station list built for large result sets
- 📻 Persistent system tray support while the app is running
- ℹ️ In-app Help section with version info and support access
- 🌐 Discord support link that opens in your browser

## Installation

1. Download the latest Windows installer from the [Releases](https://github.com/KernFerm/Radio-Player-Pro/releases/tag/V0033-Security-Update-Radio-Player-Pro) page.
2. If you already have an older version installed, uninstall the old version first.
3. Run the installer.
4. Accept the license, choose the install folder, and complete setup.
5. Open Radio Player Pro from the desktop shortcut or Start menu.

## Updating From An Older Version

- 🔄 Uninstall the older installed version before installing `v0.0.33`.
- 🧹 This helps avoid old shortcuts, stale install paths, or leftover files being reused.
- 🚀 If Windows still shows an old shortcut after uninstall, restart Windows and launch the new version from the fresh install path.

## What Gets Saved

- ❤️ Favorite stations
- ▶️ Last played station
- 🔊 Volume and mute settings
- 🎛️ Mini player visibility
- 💾 Station cache for faster browsing

## Storage Locations

- 📁 App data: `%APPDATA%\RadioPlayerPro`
- 🗂️ Cache: `%TEMP%\RadioPlayerProCache`

## Support

- ℹ️ Use the in-app Help section for version and build information.
- 💬 Join the Discord community from the app Help menu if you need help or want to report issues.

## Notes

- 📡 Radio streams and metadata come from public third-party radio directories and stations.
- ⏱️ Stream availability depends on whether the upstream station is online and working.
- 📌 The Windows tray icon may appear in the taskbar overflow area depending on your system tray settings.
